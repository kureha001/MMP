# 終了画面シーン README

## 概要
ゲーム終了時のシーンを受け持つ。

## 格納先コンテナ
シーンIDをキーに、辞書型コンテナを用いる。
| コンテナ　　  | 説明　　　　　　　　　　　　　　　　　　 |
|---------------|------------------------------------------|
| objシーン{} 　| タイトル画面・プレイ画面・終了画面で共有 |

## ライフサイクル
| タイミング | プロセス | 説明　　　　　　　　　　 |
|------------|----------|--------------------------|
| 生成     　| －　　　 | ゲーム初期化で生成　　　 |
| 生存中　　 | 移動　　 | クラス構造の概要を参照→ |
| 　　　　　 | 描画　　 | クラス構造の概要を参照→ |
| 削除   　　| －　　　 | 常に存在　　　　　　　　 |

## クラス構造
| クラス | 有無 | 概要　　　　　　　　　　　　　　　　　 |
|--------|------|----------------------------------------|
| FN移動 |　○　| 表示継続時間を管理 　　　　　　　　　　|
|　　　　|　　　| 継続時間切れで、タイトル画面へ遷移　　 |
| FN衝突 |　－　|　　　　　　　　　　　　　　　　　　　　|
| FN発射 |　－　|　　　　　　　　　　　　　　　　　　　　|
| FN描画 |　○　| 画面にゲームオーバーのメッセージを描画 |

## データ構造
| クラス | 仕様 | 情報 |
|--------|------|------|
| 本体   |  －  |  －  | 
| FN移動 |  ○  |  ○  | 
| FN描画 |  －  |  －  |
