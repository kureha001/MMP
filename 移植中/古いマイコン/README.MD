# 古いマイコンで扱うには

---
昭和時代に栄えたマイコンたちを利用する方法を解説します。
手持ちマイコンのうちRS-232Cを搭載している機種を**レベルコンバータを介して接続**します。

----
対応機種

- **NEC**
  - PC-8001mk2
  - PC-8801
  - PC-9801 (OS:MS-DOS,言語:Quick Basic/MIND)
- **富士通**
  - FM-8
- **シャープ**
  - X1
  - MZ-2500
- **日立**
  - BASIC MASTER Level3 mark2
- **統一規格**
  - MSX
  - CP/M80
