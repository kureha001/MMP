# 特殊効果オブジェクト

## 概要
取得するアイテムによって発動する効果を管理します。
効果の発動・継続・アップグレード・消失を処理します。

## 格納先コンテナ
| コンテナ | 説明　　　　　 |
|----------|----------------|
| obj背景　|単一オブジェクト|

## ライフサイクル
| タイミング | プロセス | 説明　　　　　　　　　　 |
|------------|----------|--------------------------|
| 生成     　| －　　　 | ゲーム初期化で生成　　　 |
| 生存中　　 | 移動　　 | クラス構造の概要を参照→ |
